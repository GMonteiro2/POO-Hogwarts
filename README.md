# 🧙‍♂️ POO Hogwarts — Aprenda como um Bruxo

> Aprenda Programação Orientada a Objetos em Python através do universo mágico de Harry Potter.

## ✨ Sobre o Projeto

Um site de estudos interativo que ensina os conceitos de POO usando analogias do mundo de Harry Potter. Cada conceito tem sua própria analogia mágica, exemplos de código comentados e um desafio prático no final.

## 📚 Módulos

| # | Módulo | Casa | Conceito |
|---|--------|------|----------|
| 01 | Classes & Objetos | 🔴 Grifinória | A planta baixa e a casa construída |
| 02 | Atributos & Métodos | 🔵 Corvinal | O que um bruxo tem e o que ele faz |
| 03 | Encapsulamento | 🟢 Sonserina | Protegendo os segredos do castelo |
| 04 | Herança | 🟢 Sonserina | O sangue mágico passa de geração em geração |
| 05 | Polimorfismo | 🟡 Lufa-Lufa | O mesmo feitiço, formas diferentes |
| 06 | Abstração | 🔵 Corvinal | Use a magia sem entender o feitiço |

## 🚀 Como Usar

1. Clone o repositório
```bash
git clone https://github.com/GMonteiro2/POO-Hogwarts.git
```

2. Abra o arquivo `index.html` no navegador
3. Escolha um módulo e comece a aprender!
